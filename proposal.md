THE PROJECT
We want you to design and build a lightweight drag-and-drop website builder that allows users to create and publish websites easily. Your project will later be integrated into Vera Connect and used by thousands of users.

Functional Requirements

1. Drag-and-Drop Builder
The core of the project is a drag-and-drop editor where users can create layouts visually. Users should be able to add and customize:

Text blocks (fonts, sizes, styles, colors)
Images
Videos
Buttons
Layout sections (columns, rows, grids)
The builder should be easy to use and responsive on both mobile and desktop.

2. Templates
Provide pre-built templates such as:

Student house pages
Sports team pages
Event pages
Association websites
Users should be able to customize templates freely.

3. Website Content Features
Users should be able to create pages where content can easily be updated, such as:

Photo albums
Event calendars
Contact or signup forms
Sales or promotional pages
Blogs
(Optional) Allow users to sell products or tickets via their site.
4. Publishing Options
A big plus would be enabling users to connect and purchase their own domain names. We would like you to research this area, as we currently do not have knowledge about it.

5. Privacy and Security
Because privacy is core to Vera Connect:

Websites should respect privacy-by-design principles.
Users must control their data.
Forms and personal information must be stored securely and in a GDPR-compliant manner.
An option for private/member-only pages is a plus.
6. Integration
The builder should be designed as a standalone module that can later be integrated into Vera Connect or other apps.

Other technical challenges

There are a few things we would like to have, but want you to research in more detail and propose which solutions are best and feasible. For instance,

Build a backend service that integrates a domain registrar API. After purchase, automatically create DNS records via a DNS provider like Cloudflare to attach the domain to the user’s site instance and provision SSL.
Build an AI-powered page generation service that uses an LLM (e.g., OpenAI models) to generate structured site content from user prompts. The system collects inputs (business type, goals, style), sends them to the model. Then a converter in the site builder converts this structured output into components/templates.
Technical Requirements

Your project will be build independent of our application, but we do want that you use the same stack:

React.js for web development
React Native for mobile support
Express.js backend
TypeScript
PostgreSQL database
